# Le spin

## Introduction

Le **spin** est une propriété intrinsèque des particules (électrons, protons, neutrons, photons…) : il se manifeste comme un « moment cinétique interne » et n’est pas une rotation au sens ordinaire. Il est **quantifié** (valeurs discrètes) et explique nombre de phénomènes : structure des atomes, magnétisme, IRM, RMN, spintronique.

**En résumé :**
- **Spin** : propriété quantique intrinsèque ; pas une vraie « rotation », mais un moment cinétique et magnétique.
- **Fermions** (spin 1/2) : électrons, protons, neutrons ; principe d’exclusion de Pauli.
- **Bosons** (spin entier) : photons (spin 1) ; pas d’exclusion.
- **Orientation** : dans une direction donnée, le spin prend des valeurs discrètes (ex. « haut » ou « bas » pour un électron).

## Vocabulaire utilisé

- **Spin** : propriété quantique intrinsèque des particules ; moment cinétique « interne », quantifié.
- **Fermion** : particule de spin demi-entier (ex. 1/2) ; deux fermions ne peuvent pas occuper le même état (principe de Pauli).
- **Boson** : particule de spin entier (ex. 0, 1) ; pas d’exclusion.
- **Orientation (haut/bas)** : projection du spin dans une direction ; valeurs discrètes (ex. +1/2 ou −1/2 pour l’électron).
- **Moment magnétique** : grandeur associée au spin ; à la base de la RMN et de l’IRM.
